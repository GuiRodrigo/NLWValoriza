## To start the server

```sh
yarn dev
```
## To be able to run, add the following dependencies

### Yarn add:

* express
* express-async-errors
* jsonwebtoken
* sqlite3
* uuid
* reflect-metadata
* @types/node
* bcryptjs

### Yarn add -D

* @types/bcryptjs
* @types/express
* @types/jsonwebtoken
* @types/uuid
* ts-node-dev
* typescript
